# ImgFlip API

### To run
```sh
$> cd ./api
```
```sh
$> touch .env
```
Add your ImgFlip account info to .env file
```
IMGFLIP_USERNAME=<your ImgFlip account>
IMGFLIP_PASSWORD=<your ImgFlip password>
```
Run the server
```sh
$> npm start
```
